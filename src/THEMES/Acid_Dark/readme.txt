Acid Dark
=========

A dark EdgeTX theme that uses an acid lime accent (0x7CD230). Based on the
structure of Clean Theme by Mate Soos.

Palette
-------
PRIMARY1   0xF0F0F0  near-white main text
PRIMARY2   0x121412  near-black main background
PRIMARY3   0x969696  muted / secondary text
SECONDARY1 0xF0F0F0  top-bar text
SECONDARY2 0x0C0E0C  top-bar / header background
SECONDARY3 0x545854  menu background behind tiles / dividers
FOCUS      0x7CD230  lime  - focused row (dark text drawn on top)
EDIT       0x7CD230  lime  - value being edited
ACTIVE     0x7CD230  lime  - switches on, checked boxes, active sliders
WARNING    0xFFB400  amber - warnings
DISABLED   0x5A5A5A  dimmed gray

Install
-------
Copy the Acid_Dark folder to /THEMES on the radio's SD card, then pick it
under Radio Setup > Themes.

Note
----
The light background and screenshot images from the source theme were removed.
The background_<WxH>.png files (320x240, 320x480, 480x272, 480x320, 800x480)
are solid fills in the dark background color 0x121412 (RGB 18,20,18),
matching PRIMARY2. Replace them with your own dark artwork if you want
something behind the menus.
